班級座位表產生器 (Classroom Seating Chart Generator)

這是一個專為老師設計的班級座位表排版工具。無需安裝任何軟體，網頁開啟即可使用，支援手機、平板與電腦操作。

👉 點此開始使用 (請將上方連結換成您在 GitHub Pages 設定頁面取得的實際網址)

✨ 主要功能

⚡ 快速隨機安排：一鍵打亂座位，公平公正。

🖱️ 拖曳與點擊排位：

支援拖曳交換座位。

支援點擊座位，從名單中直接選人入座 (手機操作超方便)。

🚫 鎖定特殊座位：可將損壞的桌椅或走道設為「不可用」，隨機安排時會自動避開。

👀 雙視角切換：

學生視角：黑板在上方 (適合投影給學生看)。

老師視角：黑板在下方，且左右鏡像翻轉 (模擬老師站在講台上的真實視野)。

🖨️ 列印與存檔：

支援下載圖片 (推薦)，將座位表存成高品質 PNG 圖檔。

支援直接列印 (自動轉圖列印，解決格式跑版問題)。

➕ 講台兩側座位：支援講台左右兩側的特別座/愛心座配置。

🚀 如何使用

點擊上方連結進入網頁。

若需修改學生名單，點擊「編輯學生名單」，一行一個名字貼上即可。

按下「隨機安排」生成座位。

若需微調，直接拖曳學生名牌，或點擊座位進行交換。

完成後，點擊「下載圖片」保存，或直接列印貼在講桌上。

🛠️ 技術說明

本專案為單一 HTML 檔案 (index.html)。

使用 Tailwind CSS 進行切版 (CDN)。

使用 html2canvas 進行圖片生成。
